# 一、简介

`Codex App` 是一款桌面端应用，支持并行处理多个 Codex 任务与对话，并内置 worktree 支持、自动化工作流及 Git 集成。

# 二、下载安装

`Codex App` 支持 macOS 和 Windows 两个平台。大多数功能在两个平台上均可使用，特定平台的例外情况将在相关文档中注明。

下载链接：[Codex App](https://developers.openai.com/codex/app)

安装完成后，Codex 应用的界面如下：
