Rust 的所有权系统（Ownership）是其最核心的语言特性，用于在编译时管理内存，无需依赖垃圾回收器（GC）。

所有权通过三条规则实现内存安全：

1. 每个值都有一个所有者；
2. 同一时间只能有一个所有者；
3. 所有者离开作用域时，值被自动释放。

这套机制的优势在于：避免了内存泄漏和悬垂指针，同时不带来任何运行时开销——内存管理完全在编译期完成。
