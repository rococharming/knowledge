# Update Todo

## 功能

扫描 `Claude Code/` 目录下所有 `.md` 文件，识别两类待完善笔记，更新 `Claude Code/todo.md`。

## 判断标准

1. **空文件或几乎为空** — 去除空白行后总行数 <= 1
2. **含"（待补充）"标记** — 正文中包含 `（待补充）` 或 `(待补充)`，记录所在章节标题

## 执行步骤

1. `find` 列出 `Claude Code/` 下所有 `.md` 文件（含子目录），跳过 `todo.md`
2. `Read` 读取每个文件，统计非空行数，检查是否含"（待补充）"
3. 分类写入 `Claude Code/todo.md`

## 输出格式

```markdown
# 笔记完善 Todo

## 待补充内容的笔记

### 高优先级（空文件）

- [ ] [[文件名]] — 空文件

### 中优先级（有待补充章节）

- [ ] [[文件名]]
  - [ ] 章节标题1
  - [ ] 章节标题2

## 已完善的笔记

- [x] [[文件名]]
```

## 注意事项

- 子目录文件使用 `[[子目录/文件名]]` 链接
- 若目录下无 `.md` 文件，告知用户并退出
- 覆盖更新已有 `todo.md`
- 向用户报告结果：高优先级 X 个、中优先级 Y 个、已完善 Z 个
- 使用简体中文
