解引用（Dereference）是通过引用访问其指向的底层值的操作。Rust 中既支持显式解引用，也在多种场景下提供自动解引用与自动借用。

## 显式解引用

使用 `*` 运算符显式解引用：

```rust
let x = 10;
let r = &x;
assert_eq!(*r, 10); // *r 得到底层值 10
```

对可变引用解引用后可修改底层值：

```rust
let mut x = 10;
let r = &mut x;
*r += 5; // x 变为 15
```

## 自动解引用

Rust 在以下场景自动解引用，无需手写 `*`：

### 字段访问

通过引用访问结构体字段时自动解引用：

```rust
let p = Person { name: "Tom".to_string(), age: 20 };
let p_ref = &p;
println!("{}", p_ref.name); // 等价于 (*p_ref).name
```

支持多层引用追踪：

```rust
let rr: &&Person = &(&p);
assert_eq!(rr.age, 20); // 自动追踪两层引用
```

### 方法调用中的自动借用

调用方法时 Rust 会根据方法签名自动借用：

```rust
let mut v = vec![20, 10, 30];
v.sort(); // 自动理解为 (&mut v).sort()
```

### 比较运算符

`==` 等比较运算符自动解引用到最终目标：

```rust
let rrx = &&&x;
let rry = &&&y;
assert!(rrx == rry); // 比较 x 和 y 的值
```

类型必须一致：

```rust
assert!(rrx == ry); // 错误：&&i32 与 &i32 类型不匹配
```

如需比较引用是否指向同一地址，使用 `std::ptr::eq`：

```rust
assert!(!std::ptr::eq(rx, ry));
```
