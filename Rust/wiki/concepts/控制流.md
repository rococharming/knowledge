# 控制流

Rust 提供 `if` 条件分支和 `loop` / `while` / `for` 三种循环结构，支持循环标签实现嵌套循环的精确控制。

## 条件分支：if

### if-else

```rust
let age: u8 = 10;
if age < 18 {
    println!("You are young!");
} else {
    println!("You are old!");
}
```

**注意**：

- 条件**不需要括号**（和 C/C++ 不同）
- 条件**必须是 bool 表达式**，不能用整数替代

### if-else if-else 链

```rust
let score: u32 = 70;
if score >= 90 {
    println!("Got A");
} else if score >= 80 {
    println!("Got B");
} else if score >= 70 {
    println!("Got C");
} else {
    println!("Got D");
}
```

### if 作为表达式

`if` 代码块是表达式，可用于赋值：

```rust
let x = 11;
let res = if x % 2 == 1 { "odd" } else { "even" };
```

两个分支必须返回相同类型。

## 循环

### loop — 无限循环

`loop` 循环体反复执行，直到遇到 `break`；`continue` 跳过本次剩余部分进入下一次。

```rust
loop {
    println!("running...");
    // break; // 需要某条件退出
}
```

**loop 作为表达式**：`break` 后可跟表达式作为返回值。

```rust
let mut count = 0;
let result = loop {
    count += 1;
    if count == 5 {
        break count * 2; // result = 10
    }
};
```

### while — 条件循环

每次循环前检查布尔条件：

```rust
let mut n = 10;
while n > 0 {
    println!("{}", n);
    n -= 1;
}
```

### for — 迭代循环

用于迭代范围或集合，避免手动索引：

```rust
// 遍历范围（不含结束值）
for i in 0..5 {
    println!("i = {}", i); // 0,1,2,3,4
}

// 含结束值
for i in 0..=5 {
    println!("i = {}", i); // 0,1,2,3,4,5
}

// 遍历数组
let arr = [1, 2, 3, 4, 5];
for val in arr {
    println!("val = {}", val);
}

// 反向遍历
for i in (0..=5).rev() {
    println!("i = {}", i); // 5,4,3,2,1,0
}

// 反向遍历
for i in (0..=5).rev() {
    println!("i = {}", i); // 5,4,3,2,1,0
}
```

范围语法：

- `start..end`：左闭右开 `[start, end)`
- `start..=end`：左闭右闭 `[start, end]`

## 循环标签（Loop Labels）

通过 `'标签名:` 标注循环，`break` / `continue` 可指定作用于哪一层：

```rust
'out_loop: for i in 0..3 {
    for j in 0..3 {
        if i == 1 && j == 1 {
            break 'out_loop; // 跳出外层循环
        }
        println!("i = {}, j = {}", i, j);
    }
}
```
