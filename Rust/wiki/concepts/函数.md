# 函数

Rust 是一种**表达式语言**，函数体由语句组成，并可以以一个表达式作为返回值。理解表达式与语句的区别是掌握 Rust 函数的关键。

## 表达式 vs 语句

| | 语句（Statement） | 表达式（Expression） |
|---|---|---|
| 特征 | 执行操作，不产生可用值 | 求值并产生一个值 |
| 示例 | `let a = 8;` | `5 + 6`、`a + b` |
| 赋值 | 不能将 let 赋值给其他值 | 可以作为块的返回值 |

```rust
// 错误：let 是语句，不能赋值
let b = (let a = 8);
```

## 代码块作为表达式

花括号 `{}` 包裹的代码块，如果最后一行是不带分号的表达式，则整个块的值就是该表达式的值：

```rust
let y = {
    let x = 3;
    x + 1  // 表达式，块的返回值
};
// y = 4
```

如果最后一行加了分号变成语句，块返回 `()`。

### 常见的表达式

- 函数调用
- 宏调用
- 代码块 `{}`
- `if` 表达式
- `loop` 表达式
- 字面量、运算式

## 函数定义

使用 `fn` 关键字，程序入口为 `main`：

```rust
fn add(x: i32, y: i32) -> i32 {
    x + y  // 表达式作为返回值（无分号）
}
```

### 函数签名

函数签名包括：**函数名、参数列表、返回类型**，如 `fn add(x: i32, y: i32) -> i32`。

### 函数声明顺序

**Rust 不要求先声明再定义**，函数可以写在调用之后：

```rust
fn main() {
    println!("{}", add(1, 2)); // 先调用
}

fn add(x: i32, y: i32) -> i32 { // 后定义
    x + y
}
```

## 参数

参数必须标注类型，多个参数用逗号分隔：

```rust
fn get(x: u32, y: u32) {
    println!("Got x: {x}, y: {y}");
}
```

- 定义中的参数为**形参**（parameters）
- 调用时传入的值为**实参**（arguments）
- **不会自动类型转换**：`u32` 不能自动当 `i32` 传入

## 返回值

在参数列表后用 `->` 指定返回类型：

```rust
fn add_one(x: i32) -> i32 {
    x + 1  // 表达式返回值
}

fn no_return() {       // 省略 -> ()
    println!("no return"); // 语句，隐式返回 ()
}
```

### 返回规则

- 函数体最后一行是**表达式**（无 `;`）→ 该值作为返回值
- 函数体最后一行是**语句**（有 `;`）→ 隐式返回 `()`
- 使用 `return` 关键字可**提前返回**
- 返回 `()` 时，签名可省略 `-> ()`
