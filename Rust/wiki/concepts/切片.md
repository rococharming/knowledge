切片（Slice）表示集合中的一段连续元素。日常开发中提到的"切片"通常指切片引用 `&[T]` 或 `&mut [T]`。

## 切片与切片引用

- **切片 `[T]`**：动态大小类型（DST），编译时无法确定大小，不能直接存储于变量或按值传递。
- **切片引用 `&[T]` / `&mut [T]`**：非拥有型**胖指针**，包含指向首元素的指针和元素数量。

## 创建切片

对数组或 `Vec` 使用范围语法创建切片：

```rust
let arr = [10, 20, 30, 40, 50];
let slice = &arr[1..4]; // &[20, 30, 40]
```

## 范围语法

| 写法 | 含义 |
|---|---|
| `start..end` | 从 `start` 到 `end`（不包含） |
| `..end` | 从开头到 `end`（不包含） |
| `start..` | 从 `start` 到结尾 |
| `..` | 整个范围 |

> 越界会导致 panic。

## 可变切片

```rust
let mut arr = [1, 2, 3];
let slice = &mut arr[..];
slice[1] = 20; // arr 变为 [1, 20, 3]
```

可变切片遵守可变引用规则：存在期间原数组不可访问。

## 作为函数参数

`&[T]` 可同时接收数组切片和 `Vec` 切片，比 `&Vec<T>` 更通用：

```rust
fn print_numbers(numbers: &[i32]) {
    for n in numbers { println!("{}", n); }
}

print_numbers(&arr);
print_numbers(&v);
print_numbers(&arr[1..]);
```

## 切片方法

| 方法 | 说明 |
|---|---|
| `len()` | 返回切片长度 |
| `is_empty()` | 判断是否为空 |
| `get(index)` | 安全访问，越界返回 `None` |
