结构体不仅保存数据，还可通过 `impl` 块定义关联的行为。`impl` 块内的函数分两类：方法（接收 `self` 参数）和关联函数（无 `self` 参数）。

## 方法

方法定义在 `impl` 块中，首个参数决定如何使用当前实例：

| 参数形式 | 含义 | 实例后续可用性 |
|---|---|---|
| `&self` | 共享借用，只读 | 可用 |
| `&mut self` | 可变借用，可修改 | 可用 |
| `self` | 取得所有权 | 不可用 |

```rust
impl Rectangle {
    fn area(&self) -> u32 { self.width * self.height }
}
```

`self` 实际是 `self: Self` 的简写，`Self` 代表 `impl` 后面的类型。

### 选择 self 形式的原则

- 只读取数据 → `&self`
- 需要修改数据 → `&mut self`
- 需要消费实例（如转换、销毁） → `self`

```rust
impl Counter {
    fn get(&self) -> u32 { self.value }           // 只读
    fn increase(&mut self) { self.value += 1; }   // 修改
    fn reset(self) { println!("consumed"); }       // 消费，调用后变量失效
}
```

### 方法调用中的自动借用

Rust 根据方法的 `self` 参数形式自动插入借用或移动：

```rust
counter.increase();             // 等价于 Counter::increase(&mut counter)
counter.get();                  // 等价于 Counter::get(&counter)
counter.reset();                // 等价于 Counter::reset(counter)，所有权转移
```

调用者无需手动写 `&mut counter` 等，Rust 编译器自动处理。

## 关联函数

`impl` 块中不接收 `self` 的函数称为**关联函数**，属于类型而非实例。主要用于构造实例：

```rust
impl Queue {
    fn new() -> Self {
        Self { older: Vec::new(), younger: Vec::new() }
    }
}
```

- `Self` 是当前 `impl` 类型的别名，`Self { ... }` 等价于 `Queue { ... }`
- 调用使用 `类型名::函数名`：`Queue::new()`

Rust 中构造函数通常命名为 `new`（如 `Vec::new()`、`String::new()`），也有其他命名（如 `Vec::with_capacity()`）。

## 关联常量

`impl` 块中可定义关联常量，与类型命名空间绑定：

```rust
impl Vector2 {
    const NAME: &str = "Vector2";
    const ZERO: Self = Self { x: 0.0, y: 0.0 };
    const UNIT: Self = Self { x: 1.0, y: 1.0 };
}

println!("{}", Vector2::NAME);       // 字符串常量
println!("{:?}", Vector2::ZERO);     // Self 类型常量
```

关联常量的类型不必是当前结构体类型：

```rust
impl User {
    const DEFAULT_AGE: u32 = 18; // u32，不是 User
}
```

关联常量只是定义在某类型命名空间下的常量，用于表示该类型下的固定值或常用值。

## 综合示例：Queue

```rust
struct Queue {
    older: Vec<i32>,
    younger: Vec<i32>,
}

impl Queue {
    fn new() -> Self { Self { older: Vec::new(), younger: Vec::new() } }
    fn push(&mut self, val: i32) { self.younger.push(val); }
    fn pop(&mut self) -> Option<i32> {
        if self.older.is_empty() {
            if self.younger.is_empty() { return None; }
            std::mem::swap(&mut self.older, &mut self.younger);
            self.older.reverse();
        }
        self.older.pop()
    }
    fn is_empty(&self) -> bool { self.older.is_empty() && self.younger.is_empty() }
}
```

方法选择自洽：`push`/`pop` 需修改用 `&mut self`，`is_empty` 只读用 `&self`，`new` 构造用关联函数。
