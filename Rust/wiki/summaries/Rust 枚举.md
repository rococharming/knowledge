# Rust 枚举

本文是对 Rust 枚举类型学习素材的系统梳理，涵盖枚举的两种基本形态、方法定义、标准库常用枚举（Option/Result）、富数据结构建模及内存布局。

## 核心内容

### 枚举的两种形态

- **C 风格枚举**：变体不携带数据，仅表示有限状态，支持判别值和整数转换
- **带数据枚举**：变体可携带数据，分为单元型、元组型、结构体型三种形式

### 关键知识点

- [[枚举类型]] — 枚举的定义、使用、变体形式与所有权规则
- [[Option]] 与 [[Result]] — 标准库中最常用的两个泛型枚举
- [[match]] 模式匹配 — 处理枚举不同变体的核心语法（将在后续笔记中深入）
- 枚举方法 — 通过 `impl` 块为枚举定义方法，与结构体方法语义一致

### 应用场景

- 用类型系统表达业务数据的所有可能形态（如 JSON 建模）
- 替代 null 的空值处理（`Option<T>`）
- 错误处理与结果传播（`Result<T, E>`）

### 内存特性

- C 风格枚举通常用最小整数类型存储
- 带数据枚举由标签 + 数据区域组成
- 空值优化使 `Option<&T>` 与 `&T` 同大小
