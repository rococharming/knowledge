# 一、生成随机值

## 1、概述

在Rust中，生成随机值主要依赖 `rand` crate，从新版 `rand` 的设计来看，和随机数生成最相关的几个核心概念是：

`rand::Rng` ：随机数生成器的基础能力抽象（Trait）

`rand::RngExt` ：在 `Rng` 基础上提高常用便捷方法的扩展Trait

`rand::rng()` ：获取一个具体随机数生成器实例

可以这样理解： **Rng负责定义底层能力，RngExt 负责补充常用接口，rng() 负责提供具体生成器** 。

`rand::Rng` 是一个Trait，用来描述一个类型具备生成随机数的基础能力，它定义了一组通用方法，使不同的随机数生成器拥有统一的使用方式。

在新版 `rand` 中，很多日常使用常用方法都放在了 `rand::RngExt` 中。它是一个扩展Trait，形式类似于：

```rust
pub trait RngExt: Rng {... }
```
这说明， `RngExt` 依赖于 `Rng` ，是建立在 `Rng` 基础之上的高层便捷接口。

`RngExt` 中的常见便捷方法：

`random()` ：生成任意类型的随机值

`random_range()` ：生成指定范围的随机值

`rand::rng()` **是新版rand推荐的统一入口** ，用来获取一个默认的随机数生成器实例。其特点为：

返回一个实现了 `Rng` Trait的对象

默认使用 **线程局部（thread-local）** 随机数生成器

自动初始化，无需手动设置seed

Rust为每个线程维护一个独立的RNG，每个线程拥有自己的RNG实例，不存在线程间竞争（无需加锁），因此性能更高。
## 2、生成随机数

示例：

```rust
use rand::RngExt;

fn main() {

let mut rng = rand::rng();

let n1: u8 = rng.random();

let n2: u16 = rng.random();

println!("Random u8: {}", n1);

println!("Random u16: {}", n2);

println!("Random u32: {}", rng.random::<u32>());

println!("Random i32: {}", rng.random::<i32>());

println!("Random float: {}", rng.random::<f64>());

}
```

## 3、生成范围内随机值

示例：

使用 `RngExt::random_range()` ，在半开放的 `[0, 10) ` 范围内（不包括 10）生成一个随机值。

```rust
use rand::RngExt;

fn main() {

let mut rng = rand::rng();

println!("Integer: {}", rng.random_range(0..10));

println!("Float: {}", rng.random_range(0.0..10.0));

}
```
结果：

![[Image 109.png]]

如果希望在相同范围内反复生成随机值，可以预先构造一个分布对象，例如均匀分布Uniform，然后重复采样。这样通常比每次都重新处理范围更高效。

示例：
```rust
use rand::distr::{Distribution, Uniform};

fn main() {

let mut rng = rand::rng();

// 实例化一个[1, 7)均匀分布

let die = Uniform::new(1, 7).unwrap();

loop {

let throw = die.sample(&mut rng); // 采样

println!("throw: {}", throw);

if throw == 6 {

break;

}

}

}
```
## 4、生成给定分布随机数

默认情况下，随机数在 `rand` crate中是 **均匀分布** 。 `rand_distr` crate提供了其他的分布类型。事实上， `rand_distr` 是 `rand::distr` 的超集，包含 `Normal` 在内的多种额外分布。

如果需要使用其他分布，可以首先实例化一个分布，然后在随机数生成器的帮助下，使用 `Distribution::sample` 从该分布中进行采样。

示例：

```rust
use rand_distr::{Distribution, Normal, NormalError};

fn main() -> Result<(), NormalError> {

let mut rng = rand::rng();

// 实例化一个均值为2.0，标准差为3.0的正态分布 N(μ, σ²)

let normal = Normal::new(2.0, 3.0)?;

let v = normal.sample(&mut rng);

println!("{} is from N(2, 9) distribution", v);

Ok(())

}
```
**关于更多信息，阅读** [**docs.rs/rand_distr/latest/rand_distr/**](http://docs.rs/rand_distr/latest/rand_distr/) 。
## 5、生成自定义类型随机值

随机生成一个元组 `  (i32, bool, f64)  ` 和用户定义类型为 Point 的变量。为默认分布（新版 `rand` 是 `StandardUniform` ）实现 `Distribution<T>` trait，以允许随机生成。

示例：

```rust
use rand::distr::{Distribution, StandardUniform};

use rand::{Rng, RngExt};

#[derive(Debug)]

struct Point {

x: i32,

y: i32,

}

impl Distribution<Point> for StandardUniform {

fn sample<R: Rng + ?Sized>(&self, rng: &mut R) -> Point {

Point {

x: StandardUniform.sample(rng),

y: StandardUniform.sample(rng),

}

}

}

fn main() {

let mut rng = rand::rng();

let rand_tuple: (i32, bool, f64) = rng.random();

let rand_point: Point = rng.random();

println!("Random tuple: {:?}", rand_tuple);

println!("Random point: {:?}", rand_point);

}
```
## 6、从一组字母数字字符创建随机密码

示例：

```rust
use rand::RngExt;

use rand::distr::Alphanumeric;

fn main() {

let rand_string: String = rand::rng()

.sample_iter(Alphanumeric) // 返回一个不断生成随机字母数字的迭代器

.take(30) // 只取30个

.map(char::from) // 转成字符，原来是u8

.collect();

println!("{}", rand_string);

}
```
或者可以直接使用 `Alphanumeric` 的 `sample_string` 方法生成：
```rust
use rand::distr::{Alphanumeric, SampleString};

fn main() {

let mut rng = rand::rng();

let rand_string = Alphanumeric.sample_string(&mut rng, 30);

println!("{}", rand_string);

}
```
## 7、从一组用户自定义字符创建随机密码

示例：

```rust
use rand::RngExt;

fn main() {

// 自定义字符集

const CHARSET: &[u8] = b"ABCDEFGHIJKLMNOPQRSTUVWXYZ\\

abcdefghijklmnopqrstuvwxyz\\

0123456789)(*&^%$#@!~";

// 密码长度

const PASSWORD_LEN: usize = 30; //

let mut rng = rand::rng();

let password = (0..PASSWORD_LEN)

.map(|_| {

let idx = rng.random_range(0..CHARSET.len());

CHARSET[idx] as char

})

.collect::<String>();

println!("{password}");

}
```
# 二、排序

## 1、整数Vector排序

通过切片提供的 `sort()` 方法对一个整数Vector进行升序排序。还有一种方法是使用切片的 `sort_unstable()` 方法，虽然该方法速度更快，但不保证排序前后相等元素的相对顺序。

示例：

```rust
fn main() {

let mut vec = vec![1, 5, 10, 2, 15];

vec.sort();

assert_eq!(vec, vec![1, 2, 5, 10, 15]);

}
```
## 2、浮点数Vector排序

`f32` 或 `f64` 的Vector，可以使用 `sort_by()` 方法和 `PartialOrd::partial_cmp` 对其进行排序。

示例：

```rust
fn main() {

let mut vec = vec![1.1, 1.15, 5.5, 1.123, 2.0];

vec.sort_by(|a, b| a.partial_cmp(b).unwrap());

println!("{:?}", vec);

}
```
浮点数不满足自反性，对于 `NaN` 值，不满足 `x == x` ，因此属于部分相等，只实现了 `PartialOrd` ，因此这里使用 `a.partial_cmp(b).unwrap()` 。如果返回 `None` ，则触发 `panic` ，无法排序。
## 3、结构体Vector排序

对具有name 和 age属性的 Person 结构体Vector排序，可以通过 `sort_by()` 方法自定义字段比较，也可以通过 `sort_by_key` 按照键排序，还可以按照自然顺序（先比较name，若相等再比较age）比较（不过要为Person实现 `Eq` 、 `PartialEq` 、 `Ord` 以及 `PartialOrd` ）。

示例1：

```rust
#[derive(Debug)]

struct Person {

name: String,

age: u8,

}

fn main() {

let mut people = vec![

Person::new("Zoe".to_string(), 25),

Person::new("Al".to_string(), 60),

Person::new("John".to_string(), 1),

];

people.sort_by(|a, b| a.age.cmp(&b.age) );

println!("{:#?}", people);

}
```
示例2：
```rust
#[derive(Debug, PartialEq, Eq, PartialOrd, Ord)]

struct Person {

name: String,

age: u8,

}

impl Person {

pub fn new(name: String, age: u8) -> Self {

Person {

name,

age

}

}

}

fn main() {

let mut people = vec![

Person::new("Zoe".to_string(), 25),

Person::new("Al".to_string(), 60),

Person::new("John".to_string(), 1),

];

people.sort();

println!("{:#?}", people);

}
```
